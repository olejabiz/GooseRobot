# GooseRobot
Гуськовый кран с возможностью регулировки основных узлов. Скрипты реализованы на языке Python в 3D симуляторе.
3д модель состоит из нескольких частей - балка, звенья цепи, крюк, цилиндр и основной корпус.
Есть узлы вращения, а также скрипты, написанные на языке godot для управления основными узлами вращения крана с помощью ползунков, а так же для изменения веса, размера крана, длины цепи, а также длины верхней выдвижной балки.
